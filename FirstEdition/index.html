Cyber Security and Machine Learning (First Edition)

This is the page for my new book "Cyber Security and Machine Learning" (First Edition), In Python and PyTorch.
This book covers topics at the cross-section of Cyber Security and Machine Learning (ML). Specifically, the book covers the use of machine learning for Cyber Security, as well as cyber security issues associated with AI models and systems, such as AI vulnerabilities.
Topics include: the techniques of machine learning and deep learning as applied to cyber security-related datasets and problems, adversarial machine learning, LLM and other pre-trained model vulnerabilities and defenses, and more.
